## 📌 Project Overview

This project analyzes survey responses from data professionals to understand salary trends, job roles, career transitions, satisfaction levels, and industry insights.  

The dashboard transforms raw survey data into meaningful business intelligence to uncover patterns in compensation, work-life balance, and career progression within the data industry.

---

## 📂 Dataset Overview

The dataset contains survey responses from data professionals across multiple countries.

### 🔢 Key Columns

#### 🔹 Survey Metadata
- Email  
- Date Taken (America/New_York)  
- Time Taken (America/New_York)  
- Browser  
- OS  
- City  
- Country  
- Referrer

#### 🔹 Career & Job Information

## 📊 Dashboard Components

### ✅ Survey Demographics
- Total Survey Takers  
- Average Age  
- Country Distribution  

### ✅ Average Salary by Job Title
- Data Scientist (Highest average salary)
- Data Engineer  
- Data Architect  
- Data Analyst  
- Database Developer  
- Students / Entry-Level  

### ✅ Favorite Programming Language
- Python dominates across roles  
- R follows as second preference  
- Other languages have smaller adoption  

### ✅ Difficulty Breaking Into Data
- Majority report moderate to high difficulty  
- Entry into data careers is perceived as challenging  

### ✅ Happiness Metrics
- Work/Life Balance Score  
- Salary Satisfaction Score  
- Additional satisfaction dimensions  

---

## 📈 Key Insights

- Data Scientists earn the highest average salary.
- Python is the most preferred programming language.
- Salary satisfaction is lower compared to work-life balance satisfaction.
- Many professionals transitioned into data from other careers.
- Breaking into data is perceived as moderately difficult.
